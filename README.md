## 📌 How It Works

* User enters a task in the input box
* Clicks **Add button**
* Task is added to the list
* Click on task → marks it as completed (line-through)
* Click on ❌ (cross icon) → deletes the task

---

## 💻 JavaScript Logic

* `addTask()` function creates new `<li>` elements dynamically
* Each task gets a **delete button (×)**
* Event delegation is used:

  * Click on **LI → toggle checked class**
  * Click on **SPAN → delete task**
